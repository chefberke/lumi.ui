# Usage Guide

Welcome to the **@lumi-work/lumi-ui** Usage Guide! This section will help you get started quickly with our UI components.

## Importing Components

To use a component, simply import it into your project:

<CodeBlockWrapper size="full">
```tsx
import { Button } from "@lumi-work/lumi-ui";

export default function AwesomeComponent() {
  return <Button variant="primary">Click Me</Button>;
}
```
</CodeBlockWrapper>

## Customization

You can customize components using Tailwind CSS classes or props:

<CodeBlockWrapper size="full">
```tsx
<Button className="bg-blue-500 text-white px-4 py-2 rounded-lg">Custom Button</Button>
```
</CodeBlockWrapper>
